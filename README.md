# GUI Basketball Games Display with Rust
- Made with egui
- Uses the balldontlie api to get NBA games happening today and display them with theirscore on the screen.

## Cloning
### Prerequsites - Rust
- `git clone https://github.com/Shaven-Wickramanayaka/gui_basketball.git`
- `cd gui_basketball`
- `cargo run`
